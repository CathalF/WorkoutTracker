# Phase 1: Project Setup & Navigation Shell

## Objective
Bootstrap an Expo project with TypeScript, set up tab/stack navigation, create the SQLite database schema, and seed the exercise library — so subsequent phases have a working app shell to build features into.

## Execution Context
- **Phase:** 1 of 6
- **Starting state:** Greenfield — no code exists, only `.planning/` docs
- **Mode:** yolo (execute without pausing for approval)
- **Parallelization:** false

## Context
- **Tech stack:** React Native + Expo, TypeScript, React Navigation, expo-sqlite
- **Navigation tabs:** Home, Log Workout, History, Progress
- **Database tables:** muscle_groups, exercises, workouts, sets
- **Decisions:** SQLite over AsyncStorage for relational data; local-first, no backend

## Tasks

### Task 1: Initialize Expo project
**What:** Create a new Expo project with the TypeScript template in the current directory.
**How:**
- Run `npx create-expo-app@latest . --template blank-typescript`
- Verify the project scaffolds correctly and `npx expo start` launches without errors
- Add a `.gitignore` appropriate for Expo/React Native if not already created
**Output:** Working Expo TypeScript project in the repo root

### Task 2: Install core dependencies
**What:** Install all libraries needed for Phase 1.
**How:**
- Install navigation: `npx expo install @react-navigation/native @react-navigation/bottom-tabs @react-navigation/native-stack react-native-screens react-native-safe-area-context`
- Install SQLite: `npx expo install expo-sqlite`
- Verify all packages install without conflicts
**Output:** `package.json` updated with all Phase 1 dependencies

### Task 4: Design and create SQLite schema
**What:** Create the database layer with schema for muscle groups, exercises, workouts, and sets.
**How:**
- Create `src/database/database.ts` — database initialization and schema creation
- Schema:
  ```sql
  CREATE TABLE IF NOT EXISTS muscle_groups (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL UNIQUE
  );

  CREATE TABLE IF NOT EXISTS exercises (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    name TEXT NOT NULL,
    muscle_group_id INTEGER NOT NULL,
    is_custom INTEGER NOT NULL DEFAULT 0,
    FOREIGN KEY (muscle_group_id) REFERENCES muscle_groups(id)
  );

  CREATE TABLE IF NOT EXISTS workouts (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    date TEXT NOT NULL,
    muscle_group_id INTEGER NOT NULL,
    notes TEXT,
    created_at TEXT NOT NULL DEFAULT (datetime('now')),
    FOREIGN KEY (muscle_group_id) REFERENCES muscle_groups(id)
  );

  CREATE TABLE IF NOT EXISTS sets (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    workout_id INTEGER NOT NULL,
    exercise_id INTEGER NOT NULL,
    set_number INTEGER NOT NULL,
    weight REAL NOT NULL,
    reps INTEGER NOT NULL,
    FOREIGN KEY (workout_id) REFERENCES workouts(id) ON DELETE CASCADE,
    FOREIGN KEY (exercise_id) REFERENCES exercises(id)
  );
  ```
- Use `expo-sqlite` synchronous API (SQLite.openDatabaseSync)
- Initialize database on app startup
**Output:** Database initializes on app launch with all 4 tables created

### Task 5: Seed exercise library
**What:** Populate the database with common exercises organized by muscle group.
**How:**
- Create `src/database/seed.ts` — seed function that runs on first launch (check if data exists before inserting)
- Muscle groups: Legs, Chest, Back, Shoulders, Biceps, Triceps, Core
- Exercises per group (8-12 common exercises each), for example:

  - **Chest:** Bench Press, Incline Bench Press, Dumbbell Fly, Cable Crossover, Push-Up, Decline Bench Press, Chest Dip
  - **Back:** Deadlift, Barbell Row, Pull-Up, Lat Pulldown, Seated Cable Row, T-Bar Row, Face Pull
  - **Shoulders:** Overhead Press, Lateral Raise, Front Raise, Rear Delt Fly, Arnold Press, Upright Row
  - **Biceps:** Barbell Curl, Dumbbell Curl, Hammer Curl, Preacher Curl, Concentration Curl, Cable Curl
  - **Triceps:** Tricep Pushdown, Overhead Tricep Extension, Skull Crusher, Close-Grip Bench Press, Tricep Dip, Kickback
  - **Core:** Plank, Crunch, Hanging Leg Raise, Russian Twist, Cable Woodchop, Ab Wheel Rollout
- Call seed function after database initialization in app startup
**Output:** Exercise library populated with ~50 exercises across 7 muscle groups
