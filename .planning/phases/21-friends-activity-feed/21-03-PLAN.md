  <action>
Overwrite the stub created in Plan 02 with the full implementation. Per CONTEXT.md: "Search by username/display name via Supabase profiles query" and "Manual search only -- no friend suggestions."

1. Use `GradientBackground` as wrapper. Custom glass header with back button (Ionicons `arrow-back`) + "Find Friends" title. Header uses `BlurView` from `expo-blur` for glass effect (same pattern as SettingsScreen custom header).
2. `GlassInput` for search text with a magnifying glass icon (`Ionicons` `search`). Debounce search to 500ms using a `useRef` timer pattern: clear previous timeout, set new timeout, call search on timeout.
3. On query change (min 2 chars), call `searchProfiles(query)` from friendService directly (not through context). Store results in local state. Show results in a `FlatList`.
4. Each result row in `GlassCard`: avatar image (`Image` with `borderRadius: 20` for 40x40 circle, or `Ionicons` `person-circle` placeholder if no avatar_url), `display_name` text, `bio` snippet (single line, truncated with `numberOfLines={1}`).
5. For each result, determine relationship status by checking against `useFriends()` data:
   - If result user ID is in `friendIds` -> show "Friends" text label (grayed out, disabled).
   - If result user ID matches `requester_id` or `addressee_id` in `pendingOutgoing` -> show "Requested" text label.
   - If result user ID matches `requester_id` in `pendingIncoming` -> show "Accept" GlassButton.
   - Otherwise -> show "Add" GlassButton that calls `sendRequest(userId)` from useFriends context.
6. Filter out the current user from search results: `results.filter(r => r.id !== user?.id)` using `useAuth().user`.
7. Tap on the row (not button) navigates to `UserProfile: { userId: result.id }` via `useNavigation<NativeStackNavigationProp<SocialStackParamList>>()`.
8. Loading state: `ActivityIndicator` centered while searching. Empty result state: "No users found" text centered.
9. Initial state (before typing): show instructional text "Search by name to find friends".
10. Use `useThemeControl()` for colors and isDark. Follow glass design system.
