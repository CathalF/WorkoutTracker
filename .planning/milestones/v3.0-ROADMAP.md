## Key Decisions
| Decision | Context |
|----------|---------|
| expo-blur for glass effects | BlurView from expo-blur is the core building block for frosted glass surfaces |
| Theme mode persistence via settings | User's theme preference stored in settings table under key 'theme_mode' |
| Monochrome + electric blue accent | Glass surfaces are neutral grayscale; electric blue reserved for interactive elements only |
| useThemeControl over useTheme | Screens needing isDark use useThemeControl() instead of useTheme() |
| GlassCard for exercise blocks | ActiveWorkoutScreen exercise blocks wrapped in GlassCard |
| GlassModal for all modals | All modals across all screens migrated to GlassModal |
| ScrollView paging for onboarding | No pager library needed; built-in horizontal ScrollView with pagingEnabled |
| Animated API over reanimated for onboarding | RN built-in Animated sufficient for fade+slide entrances |
| Static colors for ErrorBoundary | ErrorBoundary wraps ThemeProvider so uses hardcoded colors |
