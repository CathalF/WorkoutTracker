3. **Workout Logging Flow** — Split selection, exercise picker, fast set logging with quick-increment buttons
4. **Workout History** — Chronological workout list with date grouping, filter chips, detail view with inline editing

## Key Decisions
| Decision | Context |
|----------|---------|
| expo-sqlite sync API | Used openDatabaseSync + execSync for simpler init flow |
| Slug workout-tracker | create-expo-app requires URL-friendly names |
| In-memory workout state | Workout data kept in component state during session, only persisted on finish |
| Exercise selection via nav params | ExercisePicker passes selectedExercise back via route params |
| react-native-gifted-charts | Zero native modules, Expo Go compatible |
